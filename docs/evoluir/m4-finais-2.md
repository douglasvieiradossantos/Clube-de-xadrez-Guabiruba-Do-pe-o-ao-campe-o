# M4 — Finais práticos (torres)

## Objetivo
Entender regras práticas para ganhar/segurar finais de torre.

## Princípios básicos
- Torre ativa é tudo (atacar peões por trás)
- Rei centralizado ajuda
- Passados devem avançar com apoio

## Erros comuns
- Torre passiva defendendo peões sem atividade
- Deixar o rei longe da ação
