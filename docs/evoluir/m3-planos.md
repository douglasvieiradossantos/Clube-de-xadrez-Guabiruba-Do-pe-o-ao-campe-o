# M3 — Planos no meio-jogo

## Objetivo
Parar de jogar “lance solto” e começar a jogar com plano.

## Perguntas-guia
- Qual é a pior peça minha? Como melhorar?
- Onde estão as casas fracas do adversário?
- Há coluna aberta para minhas torres?
- Vou atacar o rei ou ganhar no final?

## Planos comuns
- Dobrar torres em coluna aberta
- Melhorar cavalo para casa forte
- Trocar peças quando estiver ganhando material/estrutura
