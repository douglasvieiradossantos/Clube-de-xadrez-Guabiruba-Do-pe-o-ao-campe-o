# M4 — Finais essenciais (rei e peões)

## Objetivo
Aprender finais básicos de **rei e peões** para não “entregar” vitórias.

## Conceitos
- Rei é peça forte no final.
- "Oposição" (ideia): colocar os reis frente a frente para ganhar casas.
- Promover peão: criar um peão passado e escoltar.

## Exercícios (coloque depois)
1) Levar o rei para ajudar um peão a promover.
2) Impedir o rei adversário de entrar.
